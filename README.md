# scleaner

This app helps formatting and exporting university schedule of Gdańsk Technical University.
It get's unformatted **.csv** file (copied from [class schedule page](https://sis.eti.pg.edu.pl/)) as an input and provides some useful features and a preview of the final result.

## Feature list

- Merge duplicate classes into one.
- Toggle Lectures visibility.
- Show optional classes for one subgroup only.
- Choose colors for different types of classes.
- Save current schedule as a file (for further modifications).
- Render schedule as an image (adapted for mobile screen).
